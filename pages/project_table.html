Projet table interactive

Une table tactile multimédia

Description

Réalisée dans le cadre du cours Nouvelles technologies du DEC informatique 420.A2, cette table interactive m'a permis d'explorer les technologies de reconnaissance d'objets tactiles.

Ce projet se divise en deux parties. La première consiste à la conception et fabrication d'une table de salon interactive intégrant un écran tactile. La seconde, consiste en la conception d'un programme de reconnaissance tactile d'objets fut développé avec WPF en C#.

Techniques utilisées

  • C#
  • WPF
  • Python
  • Raspberry pi
  • IT
  • Menuiserie
