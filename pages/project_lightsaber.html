Projet sabre laser

Replique de sabre laser

Description

Ce projet est une tentative de reproduction du célebre outil de combat fictif de la franchise Star Wars. Il prend son origine dans l'optique qu'une reproduction peut s'avouer extrêmement dispendieuse et donc, ce projet fut realisé en gardant un budget.

Techniques utilisées

  • C#
  • C++
  • C
  • WPF
  • Unity
  • Java
